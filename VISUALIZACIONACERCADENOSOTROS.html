¿Quién soy yo?

Soy una estudiante de la UAM-I. Estoy terminando mi licenciatura.

Este proyecto, Sistema Web para Ingeniería de Software, es una herramienta educativa diseñada para estudiantes de nivel secundaria. Su objetivo principal es facilitar el aprendizaje de conceptos fundamentales de desarrollo web, como HTML, CSS, JavaScript y PHP, a través de un libro digital interactivo.

El sistema web permite a los estudiantes:

  • ⭐Aprender a crear páginas web desde cero.
  • ⭐Explorar ejemplos prácticos de código.
  • ⭐Interactuar con un traductor integrado para mejorar su comprensión.
  • ⭐Acceder a una base de datos de recursos educativos.

Este proyecto fue desarrollado por Gaby C. Esto fue parte de su esfuerzo para facilitar el acceso a la educación tecnológica y fomentar el interés en la ingeniería de software desde una edad temprana, en este caso a los estudintes de nivel secundaria.

Tecnologías Utilizadas

HTML CSS JavaScript PHP
ACERCA DEL SISTEMA WEB

Es un sistema web del desarrollo de contenido didáctico para promover el desarrollo web de la Ingeniería de Software a nivel secundaria. Se encuentra concentrada toda la información, esta estará en diferentes tipos de contenido (documentos, códigos, ejercicios, imágenes, videos, tutoriales, encuestas, entre otros), esto permitirá y facilitará la gestión eficiente del contenido educativo, también estará disponible las 24/7, podrá ser accesible a través de diversos dispositivos electrónicos (computadoras, celulares, tabletas, etc.).
